Quick Wins

Good habits to
start today

1 🔍

Fact-check AI outputs

Never publish AI-generated content without verifying key claims. Confidence ≠ accuracy.

2 🔐

Strong, unique passwords

Use a password manager. Never reuse passwords across work and personal accounts.

3 ☁️

Save to OneDrive

Files saved locally can be lost. OneDrive gives you version history and recovery.

4 ⚠️

Never upload pupil data to AI

Personal data about students must never be entered into public AI tools like ChatGPT.

5 ✉️

Pause before you click

Phishing emails create urgency. Always verify unexpected requests before acting.

6 🧠

Be transparent about AI

If AI helped you create something, say so. Model good digital citizenship for students.
